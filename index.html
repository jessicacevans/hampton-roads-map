Methods & Data Sources

  • Boundaries: U.S. Census Bureau TIGER/Line geographic identifiers, decoded from us-atlas TopoJSON (Census TIGER/Line derived). Embedded as GeoJSON; no runtime API calls.
  • Population: 2020 Census P.L. 94-171 Redistricting Data Summary File.
  • Land Area: Census TIGER/Line geographic identifiers (square miles).
  • Basemap: © OpenStreetMap contributors, ODbL.
  • Pipeline: ACS 5-Year estimates joined to TIGER boundaries on GEOID (text field, leading zeros preserved). Python / GeoPandas workflow.
